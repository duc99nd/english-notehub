# Subjunctive Mood & Formal Demands

In technical documentation, pull request reviews, and emails to clients or managers, you often need to express requirements, recommendations, or urgent requests. To sound professional and grammatically correct, you must use the **Subjunctive Mood**.

---

## 1. What is the Subjunctive Mood?

The subjunctive mood is used when expressing a wish, demand, suggestion, or requirement. 
- **The Golden Rule:** The verb in the `that` clause is **always in its base form** (bare infinitive). It does not change based on the subject (no `-s`, `-es`, or past tense).

- ❌ *Wrong:* "It is important that he **updates** the database config."
- 🟢 *Correct:* **"It is important that he update the database config."**
- ❌ *Wrong:* "We recommend that the team **deploys** the hotfix now."
- 🟢 *Correct:* **"We recommend that the team deploy the hotfix now."**

---

## 2. Common Adjectives Triggering the Subjunctive

Use this formula: `It is + Adjective + that + Subject + Base Verb`

- **crucial / critical / essential (cực kỳ quan trọng):**
  - **"It is critical that the server remain online during the migration."** (Note: *remain*, not *remains*).
- **important / necessary (quan trọng / cần thiết):**
  - **"It is necessary that the user reset their password after the first login."** (Note: *reset*, not *resets*).
- **mandatory / required (bắt buộc):**
  - **"It is mandatory that every developer write unit tests for their code."**

---

## 3. Common Verbs Triggering the Subjunctive

Use this formula: `Subject + Verb + that + Subject + Base Verb`

- **recommend / suggest (đề xuất/khuyên):**
  - **"I recommend that Huy review the security policy."** (Note: *review*, not *reviews*).
  - **"The architect suggested that we database-shard the database next year."**
- **demand / insist (yêu cầu/quả quyết):**
  - **"The client insists that the app support offline mode."** (Note: *support*, not *supports*).
- **require / ask (đòi hỏi):**
  - **"The system requires that the admin approve the transaction."**

---

## 4. Negative Subjunctive (Not + Base Verb)

To make a subjunctive sentence negative, simply put **not** directly before the base verb (do **NOT** use *does not*, *do not*, or *did not*).

- ❌ *Wrong:* "It is vital that we don't expose the API key."
- 🟢 *Correct:* **"It is vital that we not expose the API key."**
- 🟢 **"I recommend that the script not run automatically."** (Instead of: *should not run* or *doesn't run*).

---

## 5. Summary Cheat Sheet

| Trigger Phrase | Example Sentence | Meaning |
| :--- | :--- | :--- |
| **It is crucial that...** | "It is crucial that the lead developer **sign** off on this." | Trưởng nhóm cần ký duyệt cái này. |
| **I recommend that...** | "I recommend that he **not pull** the changes yet." | Tôi khuyên anh ấy chưa nên pull code mới về. |
| **The client asks that...**| "The client asks that the design **be** simplified." | Khách hàng yêu cầu thiết kế được đơn giản hóa. |
