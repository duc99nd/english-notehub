# Conjunctions & Transition Words

Transitions and conjunctions are essential for joining ideas together logically. However, using them with incorrect punctuation or sentence structures is one of the most common grammatical errors in professional IT communication.

---

## 1. Conjunctions vs. Transition Words

- **Conjunctions (Liên từ):** Words like *but*, *so*, *and*, *or*. They connect two clauses **inside a single sentence**. They are usually preceded by a comma.
- **Transition Words (Từ nối chuyển tiếp):** Words like *however*, *therefore*, *moreover*, *furthermore*. They connect **two separate sentences** and are always followed by a comma.

---

## 2. Hack 1: "But" vs. "However" (Contrasting Ideas)

- **but:** Use inside one sentence. Put a comma before it.
  - 🟢 **"The query is fast, but it uses too much memory."**
- **however:** Starts a new sentence. Put a period before it and a comma after it.
  - 🟢 **"The query is fast. However, it uses too much memory."**
  - ❌ *Wrong:* "The query is fast, however, it uses too much memory." (This is a run-on sentence error).

### 💡 10 Practical Examples:

1.  **"We wanted to deploy yesterday, but the staging tests failed."**
2.  **"We wanted to deploy yesterday. However, the staging tests failed."**
3.  **"Huy finished the frontend designs, but the client hasn't approved them yet."**
4.  **"Huy finished the frontend designs. However, the client hasn't approved them yet."**
5.  **"The setup is simple, but the documentation is a bit outdated."**
6.  **"The setup is simple. However, the documentation is a bit outdated."**
7.  **"Docker works well on Linux, but it can be slow on macOS."**
8.  **"Docker works well on Linux. However, it can be slow on macOS."**
9.  **"The API handles read requests quickly, but write operations are slow."**
10. **"The API handles read requests quickly. However, write operations are slow."**

---

## 3. Hack 2: "So" vs. "Therefore" (Showing Results)

- **so:** Use inside one sentence to connect an action and its result.
  - 🟢 **"The staging database crashed, so we paused testing."**
- **therefore:** Starts a new sentence to explain a formal consequence.
  - 🟢 **"The staging database crashed. Therefore, we paused testing."**
  - 🟢 **"The staging database crashed; therefore, we paused testing."** (Using a semicolon is also correct).

### 💡 10 Practical Examples:

1.  **"The backup failed last night, so we ran it manually this morning."**
2.  **"The backup failed last night. Therefore, we ran it manually this morning."**
3.  **"The server run-time is too high, so we need to upgrade the instances."**
4.  **"The server run-time is too high; therefore, we need to upgrade the instances."**
5.  **"Huy was on leave, so I took over the authentication task."**
6.  **"Huy was on leave. Therefore, I took over the authentication task."**
7.  **"The package contains security flaws, so we downgraded it immediately."**
8.  **"The package contains security flaws. Therefore, we downgraded it immediately."**
9.  **"The database table was locked, so all insert queries timed out."**
10. **"The database table was locked; therefore, all insert queries timed out."**

---

## 4. Hack 3: "Because" vs. "Due to / Because of" (Showing Causes)

- **because:** Must be followed by a full clause (`Subject + Verb`).
  - 🟢 **"We had a delay because the API credentials expired."**
  - ❌ *Wrong:* "We had a delay because of the API credentials expired."
- **due to / because of:** Must be followed by a Noun Phrase or V-ing.
  - 🟢 **"We had a delay due to the expired API credentials."**
  - 🟢 **"We had a delay because of the expired API credentials."**
  - ❌ *Wrong:* "We had a delay due to the API credentials expired."

### 💡 10 Practical Examples:

1.  **"The API request timed out because the database was locked."**
2.  **"The API request timed out due to the locked database."**
3.  **"We resolved the issue because Huy refactored the query."**
4.  **"We resolved the issue because of Huy's query refactoring."**
5.  **"The pipeline broke because a configuration file was missing."**
6.  **"The pipeline broke due to a missing configuration file."**
7.  **"The build failed because node modules were outdated."**
8.  **"The build failed because of outdated node modules."**
9.  **"Our traffic spiked because the marketing campaign launched."**
10. **"Our traffic spiked due to the launch of the marketing campaign."**

---

## 5. Quick Connector Summary

| Relationship | Conjunction (Use inside one sentence) | Transition (Start a new sentence) | Cause/Effect (Uses Nouns) |
| :--- | :--- | :--- | :--- |
| **Contrast (Đối lập)** | `, but` | `. However,` | `Despite / In spite of` |
| **Result (Kết quả)** | `, so` | `. Therefore,` / `. As a result,` | |
| **Cause (Nguyên nhân)** | `because` / `since` / `as` | | `due to` / `because of` |
| **Addition (Thêm ý)** | `, and` | `. Also,` / `. In addition,` | |
