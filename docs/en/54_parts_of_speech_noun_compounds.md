# Word Families & Noun Compounds

Mixing up Nouns, Verbs, and Adjectives when speaking can make updates sound awkward. This document details how to choose the correct word form in a sentence, provides the core IT word families, and explains the rule of noun compounds (noun-noun modifiers).

---

## 1. Core IT Word Families

Always learn the three most common forms of core IT concepts together:

| Verb (Action) | Noun (Concept/Object) | Adjective (State/Quality) |
| :--- | :--- | :--- |
| **optimize** (tối ưu hóa) | **optimization** (sự tối ưu) | **optimized** (đã tối ưu) |
| **secure** (bảo mật hóa) | **security** (sự bảo mật) | **secured / secure** (an toàn/được bảo mật) |
| **implement** (triển khai) | **implementation** (sự triển khai) | **implemented** (đã triển khai) |
| **configure** (cấu hình) | **configuration** (file/sự cấu hình) | **configured** (đã cấu hình) |
| **integrate** (tích hợp) | **integration** (sự tích hợp) | **integrated** (đã tích hợp) |
| **automate** (tự động hóa) | **automation** (sự tự động) | **automated** (tự động) |
| **standardize** (tiêu chuẩn hóa) | **standardization** (sự tiêu chuẩn) | **standardized** (đã tiêu chuẩn hóa) |
| **validate** (xác thực dữ liệu) | **validation** (sự xác thực) | **valid** (hợp lệ) |
| **encrypt** (mã hóa) | **encryption** (sự mã hóa) | **encrypted** (đã mã hóa) |
| **deploy** (triển khai code) | **deployment** (sự deploy) | **deployed** (đã deploy) |
| **migrate** (di chuyển dữ liệu) | **migration** (sự di chuyển) | **migrated** (đã di chuyển) |

## 2. Choosing the Right Word Form

To speak quickly without hesitating, use these structural markers to know which word form is correct.

### 💡 Hack 1: The Verb Position
Use a **Verb** immediately after helper verbs (`will/should/can/could`) or infinitives (`need to/plan to/decide to`).

*   **10 Practical Examples:**
    1.  **"We plan to optimize our database queries."**
    2.  **"This tool can automate the build process."**
    3.  **"We decided to deploy the hotfix today."**
    4.  **"We must validate the user input on the server."**
    5.  **"The script will encrypt all backup files."**
    6.  **"We should standardize our API response formats."**
    7.  **"We management managed to configure the firewall rules."**
    8.  **"We need to integrate the third-party payment portal."**
    9.  **"You should secure the API endpoints with token validation."**
    10. **"We plan to migrate our assets to AWS RDS next week."**

---

### 💡 Hack 2: The Noun Position
Use a **Noun** immediately after prepositions (`for/by/about/with`) or possessive adjectives (`our/my/their`).

*   **10 Practical Examples:**
    1.  **"This library is used for user authentication."**
    2.  **"Let me check our staging configuration."**
    3.  **"The team is focusing on database optimization."**
    4.  **"We resolved the issue by performing a clean deployment."**
    5.  **"The client requested a demo of the new integration."**
    6.  **"We need to double-check the security of the server."**
    7.  **"This schema migration will take about two hours."**
    8.  **"The middleware handles input validation."**
    9.  **"I'm responsible for the automation of test runs."**
    10. **"We are standardizing our setup configuration."**

---

### 💡 Hack 3: The Adjective Position
Use an **Adjective** immediately before a noun or after linking verbs (`is/are/look/feel/seem`).

*   **10 Practical Examples:**
    1.  **"This is an automated testing tool."**
    2.  **"Our servers look secure."**
    3.  **"Please ensure the database token remains valid."**
    4.  **"All passwords are encrypted before storage."**
    5.  **"The configuration is ready for the deployed build."**
    6.  **"This is our standardized architecture pattern."**
    7.  **"We need an optimized query to load the feed."**
    8.  **"The integrated payment service works perfectly."**
    9.  **"We have a fully configured dev environment."**
    10. **"Is the migrated database stable now?"**

## 3. The Noun Compound Hack

In tech, we frequently group two nouns together to describe one thing (e.g., database connection, user profiles).

### 💡 Hack 4: The First Noun is Always Singular
When a noun modifies another noun, **the first noun acts like an adjective**. Adjectives in English cannot be plural. Therefore, never add an "s" to the first noun, even if the second noun is plural.

*   ❌ *Wrong:* databases connections, users profiles, APIs keys, logs files.
*   🟢 *Correct:* **database connections**, **user profiles**, **API keys**, **log files**.

### 💡 10 Technical Noun Compounds to Master:

1.  **"We need to rotate our API keys immediately."** (Not: *APIs keys*)
2.  **"Check the server logs for database errors."** (Not: *databases errors*)
3.  **"Huy is working on user accounts management."** (Not: *users accounts*)
4.  **"Please review our testing procedures for network switches."** (Not: *networks switches*)
5.  **"The system crashed due to memory leaks."** (Not: *memories leaks*)
6.  **"I am configuring the container image settings."** (Not: *containers images*)
7.  **"We need to clean up these error messages."** (Not: *errors messages*)
8.  **"The project requires two software engineers."** (Not: *softwares engineers*)
9.  **"The user session stores browser cookies."** (Not: *browsers cookies*)
10. **"Let's modify the account settings."** (Not: *accounts settings*)
