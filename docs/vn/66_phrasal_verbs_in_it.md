# Cụm Động Từ (Phrasal Verbs) Trong IT (Daily English)

Phrasal verb là các cụm từ gồm 2-3 từ (động từ + giới từ/trạng từ) có nghĩa riêng biệt, khác hẳn nghĩa gốc của từng từ. Người bản xứ dùng chúng liên tục trong cuộc họp, tin nhắn Slack và code review. Nắm vững các phrasal verb trong IT sẽ giúp tiếng Anh của bạn nghe tự nhiên và trôi chảy hơn rất nhiều.

---

## 1. Phát Triển & Viết Code

### 💡 10 Phrasal Verb Cần Biết:

1.  **set up** — Cấu hình hoặc chuẩn bị một thứ gì đó để sử dụng.
    - *"Can you **set up** the local development environment?"* *(Bạn có thể cài đặt môi trường dev ở local được không?)*
2.  **spin up** — Khởi động, dựng lên (server, container, instance).
    - *"Let me **spin up** a new Docker container for testing."* *(Để tôi dựng một Docker container mới để test)*
3.  **tear down** — Tháo dỡ, gỡ bỏ hạ tầng.
    - *"We need to **tear down** the staging environment after the demo."* *(Chúng ta cần gỡ bỏ môi trường staging sau buổi demo)*
4.  **hook up** — Kết nối hai hệ thống hoặc dịch vụ.
    - *"I **hooked up** the frontend to the new REST API."* *(Tôi đã kết nối frontend với API REST mới)*
5.  **break down** — Chia nhỏ thành các phần nhỏ hơn; cũng có nghĩa là ngừng hoạt động.
    - *"Let's **break down** this user story into subtasks."* *(Hãy chia nhỏ user story này thành các task con)*
    - *"The server **broke down** during peak traffic."* *(Server bị sập trong giờ cao điểm)*
6.  **plug in** — Cắm vào, thêm một thành phần hoặc module.
    - *"We just need to **plug in** the payment gateway module."* *(Chúng ta chỉ cần cắm thêm module cổng thanh toán vào thôi)*
7.  **opt in / opt out** — Chọn tham gia / chọn không tham gia.
    - *"Users can **opt out** of email notifications in their settings."* *(Người dùng có thể chọn tắt thông báo email trong phần cài đặt)*
8.  **figure out** — Tìm ra, giải quyết một vấn đề.
    - *"I need to **figure out** why the API returns a 403 error."* *(Tôi cần tìm ra tại sao API trả về lỗi 403)*
9.  **comment out** — Vô hiệu hóa code bằng cách đưa vào dấu comment.
    - *"I **commented out** the debug logs before pushing to production."* *(Tôi đã comment out các dòng log debug trước khi push lên production)*
10. **refactor out** — Loại bỏ code thừa thông qua việc refactor.
    - *"We should **refactor out** the duplicated validation logic."* *(Chúng ta nên refactor bỏ đi logic validate bị trùng lặp)*

## 2. Deployment & Vận Hành

### 💡 10 Phrasal Verb Cần Biết:

1.  **roll back** — Đưa về phiên bản trước đó.
    - *"The hotfix failed. We need to **roll back** to version 1.4.2."* *(Bản hotfix thất bại. Chúng ta cần rollback về phiên bản 1.4.2)*
2.  **roll out** — Triển khai, phát hành phiên bản mới.
    - *"We will **roll out** the new feature to 10% of users first."* *(Chúng ta sẽ triển khai tính năng mới cho 10% người dùng trước)*
3.  **scale up / scale out** — Tăng tài nguyên (theo chiều dọc/chiều ngang).
    - *"We need to **scale up** the database server to handle more queries."* *(Chúng ta cần scale up server database để xử lý nhiều query hơn)*
4.  **shut down** — Dừng, tắt một dịch vụ hoặc server.
    - *"Please **shut down** the staging server before the maintenance window."* *(Vui lòng tắt server staging trước khung giờ bảo trì)*
5.  **back up** — Sao lưu dữ liệu để bảo vệ.
    - *"Make sure you **back up** the database before running the migration."* *(Đảm bảo bạn đã backup database trước khi chạy migration)*
6.  **switch over** — Chuyển đổi từ hệ thống này sang hệ thống khác.
    - *"We will **switch over** to the new CDN provider at midnight."* *(Chúng ta sẽ chuyển sang nhà cung cấp CDN mới lúc nửa đêm)*
7.  **phase out** — Dần dần ngưng sử dụng.
    - *"We are **phasing out** support for Node 14 by the end of Q3."* *(Chúng ta đang dần khai tử việc hỗ trợ Node 14 trước cuối Q3)*
8.  **boot up** — Khởi động máy hoặc hệ thống.
    - *"The server takes 45 seconds to **boot up** after a restart."* *(Server mất 45 giây để khởi động lại sau khi restart)*
9.  **log in / log out** — Đăng nhập / Đăng xuất.
    - *"Users are unable to **log in** after the latest deployment."* *(Người dùng không thể đăng nhập sau đợt deploy mới nhất)*
10. **patch up** — Sửa tạm thời.
    - *"Let's **patch up** the security hole tonight and do a proper fix next sprint."* *(Hãy vá tạm lỗ hổng bảo mật tối nay và sửa chính thức vào sprint sau)*

## 3. Giao Tiếp & Họp Hành

### 💡 10 Phrasal Verb Cần Biết:

1.  **follow up** — Theo dõi tiến độ sau khi đã thảo luận.
    - *"I will **follow up** with the client about the API credentials."* *(Tôi sẽ liên hệ lại khách hàng về thông tin API)*
2.  **bring up** — Đưa ra một chủ đề trong cuộc họp.
    - *"I want to **bring up** the database bottleneck in today's standup."* *(Tôi muốn đề cập đến vấn đề nghẽn DB trong buổi standup hôm nay)*
3.  **point out** — Chỉ ra, nhấn mạnh.
    - *"Huy **pointed out** that the SSL certificate expires next week."* *(Huy đã chỉ ra rằng chứng chỉ SSL sẽ hết hạn vào tuần tới)*
4.  **go over** — Xem lại, rà soát.
    - *"Let's **go over** the deployment checklist before we push to production."* *(Hãy rà soát lại checklist deploy trước khi push lên production)*
5.  **wrap up** — Kết thúc, chốt lại.
    - *"Let's **wrap up** this meeting. We've covered all the action items."* *(Hãy kết thúc cuộc họp này. Chúng ta đã bàn xong tất cả các action item)*
6.  **circle back** — Quay lại một chủ đề sau.
    - *"Let's **circle back** on the caching strategy after lunch."* *(Hãy quay lại chủ đề chiến lược caching sau giờ trưa)*
7.  **sign off** — Phê duyệt chính thức.
    - *"The client needs to **sign off** on the design before we start coding."* *(Khách hàng cần phê duyệt bản thiết kế trước khi chúng ta bắt đầu code)*
8.  **hold off** — Hoãn lại, trì hoãn.
    - *"Let's **hold off** on the migration until the performance tests pass."* *(Hãy hoãn lại việc migration cho đến khi test hiệu năng pass)*
9.  **run by** — Chia sẻ ý tưởng để xin ý kiến phản hồi.
    - *"Can I **run** this architecture proposal **by** you before the meeting?"* *(Tôi có thể trình bày bản đề xuất kiến trúc này cho bạn trước cuộc họp không?)*
10. **catch up** — Cập nhật lại thông tin đã bỏ lỡ.
    - *"I was on leave. Can we **catch up** on what happened during the sprint?"* *(Tôi vừa nghỉ phép. Chúng ta có thể cập nhật lại những gì xảy ra trong sprint không?)*

| **roll back** | Quay lại phiên bản cũ | "**Roll back** to the last release" |
| **roll out** | Triển khai | "**Roll out** the feature flag" |
| **scale up** | Thêm tài nguyên | "**Scale up** the database server" |
| **back up** | Sao lưu | "**Back up** the production DB" |
| **follow up** | Theo dõi tiến độ | "**Follow up** on the PR review" |
| **bring up** | Nêu chủ đề | "**Bring up** the issue in standup" |
| **wrap up** | Kết thúc | "**Wrap up** the retro meeting" |
| **figure out** | Tìm ra/Giải quyết | "**Figure out** the root cause" |
