# Liên từ & Từ nối Chuyên nghiệp (Daily English)

Liên từ và từ nối là những yếu tố bắt buộc phải dùng để liên kết các ý tưởng một cách logic. Tuy nhiên, việc đặt dấu câu sai hoặc viết sai cấu trúc câu khi dùng chúng lại là những lỗi phổ biến nhất trong giao tiếp tiếng Anh chuyên ngành IT.

---

## 1. Phân biệt Liên từ (Conjunctions) và Từ nối (Transition Words)

- **Liên từ (Conjunctions):** Gồm các từ như *but*, *so*, *and*, *or*. Chúng làm nhiệm vụ nối hai mệnh đề **nằm trong cùng một câu**. Trước liên từ thường có dấu phẩy.
- **Từ nối chuyển tiếp (Transition Words):** Gồm các từ như *however*, *therefore*, *moreover*, *furthermore*. Chúng làm nhiệm vụ nối **hai câu riêng biệt** và luôn có dấu phẩy đi ngay sau.

---

## 2. Mẹo 1: Phân biệt "But" và "However" (Ý tương phản)

- **but:** Dùng để nối 2 vế trong cùng 1 câu. Luôn đặt dấu phẩy trước `but`.
  - 🟢 **"The query is fast, but it uses too much memory."** *(Câu query chạy nhanh, nhưng nó tốn quá nhiều bộ nhớ).*
- **however:** Dùng để bắt đầu một câu độc lập mới. Luôn đặt dấu chấm trước `however` và dấu phẩy sau `however`.
  - 🟢 **"The query is fast. However, it uses too much memory."**
  - ❌ *Sai:* "The query is fast, however, it uses too much memory." *(Lỗi viết liền hai câu độc lập chỉ bằng dấu phẩy).*

### 💡 10 Ví dụ thực tế:

1.  **"We wanted to deploy yesterday, but the staging tests failed."** *(Chúng tôi muốn deploy hôm qua, nhưng việc chạy test staging bị lỗi).*
2.  **"We wanted to deploy yesterday. However, the staging tests failed."**
3.  **"Huy finished the frontend designs, but the client hasn't approved them yet."** *(Huy đã làm xong giao diện frontend, nhưng khách hàng vẫn chưa phê duyệt).*
4.  **"Huy finished the frontend designs. However, the client hasn't approved them yet."**
5.  **"The setup is simple, but the documentation is a bit outdated."** *(Cài đặt rất đơn giản, nhưng tài liệu hướng dẫn hơi cũ).*
6.  **"The setup is simple. However, the documentation is a bit outdated."**
7.  **"Docker works well on Linux, but it can be slow on macOS."** *(Docker chạy tốt trên Linux, nhưng nó có thể bị chậm trên macOS).*
8.  **"Docker works well on Linux. However, it can be slow on macOS."**
9.  **"The API handles read requests quickly, but write operations are slow."** *(API xử lý các request đọc rất nhanh, nhưng các thao tác ghi lại chậm).*
10. **"The API handles read requests quickly. However, write operations are slow."**

## 3. Mẹo 2: Phân biệt "So" và "Therefore" (Kết quả)

- **so:** Dùng trong 1 câu để nối hành động và kết quả trực tiếp của nó.
  - 🟢 **"The staging database crashed, so we paused testing."** *(Database staging bị lỗi, nên chúng tôi đã tạm dừng việc chạy test).*
- **therefore:** Bắt đầu một câu mới hoặc đứng sau dấu chấm phẩy để giải thích kết luận logic một cách trang trọng.
  - 🟢 **"The staging database crashed. Therefore, we paused testing."**
  - 🟢 **"The staging database crashed; therefore, we paused testing."** *(Sử dụng dấu chấm phẩy cũng hoàn toàn chính xác).*

### 💡 10 Ví dụ thực tế:

1.  **"The backup failed last night, so we ran it manually this morning."** *(Bản backup đêm qua bị lỗi, nên chúng tôi đã chạy lại thủ công sáng nay).*
2.  **"The backup failed last night. Therefore, we ran it manually this morning."**
3.  **"The server run-time is too high, so we need to upgrade the instances."** *(Thời gian chạy server quá cao, nên chúng ta cần nâng cấp cấu hình máy ảo).*
4.  **"The server run-time is too high; therefore, we need to upgrade the instances."**
5.  **"Huy was on leave, so I took over the authentication task."** *(Huy nghỉ phép, nên tôi đã nhận lại task xử lý xác thực).*
6.  **"Huy was on leave. Therefore, I took over the authentication task."**
7.  **"The package contains security flaws, so we downgraded it immediately."** *(Thư viện chứa lỗ hổng bảo mật, nên chúng tôi đã hạ cấp nó ngay lập tức).*
8.  **"The package contains security flaws. Therefore, we downgraded it immediately."**
9.  **"The database table was locked, so all insert queries timed out."** *(Bảng cơ sở dữ liệu bị khóa, nên tất cả các câu query insert đều bị timeout).*
10. **"The database table was locked; therefore, all insert queries timed out."**

## 4. Mẹo 3: Phân biệt "Because" và "Due to / Because of" (Nguyên nhân)

- **because:** Theo sau bắt buộc phải là một mệnh đề đầy đủ (`Chủ ngữ + Động từ`).
  - 🟢 **"We had a delay because the API credentials expired."** *(Chúng tôi bị chậm tiến độ vì thông tin tài khoản API hết hạn).*
- **due to / because of:** Theo sau bắt buộc phải là một cụm danh từ (Noun Phrase) hoặc động từ dạng V-ing.
  - 🟢 **"We had a delay due to the expired API credentials."** *(Chúng tôi bị chậm tiến độ do thông tin tài khoản API hết hạn).*
  - 🟢 **"We had a delay because of the expired API credentials."**

### 💡 10 Ví dụ thực tế:

1.  **"The API request timed out because the database was locked."** *(API request bị timeout vì database đã bị khóa).*
2.  **"The API request timed out due to the locked database."** *(API request bị timeout do database bị khóa).*
3.  **"We resolved the issue because Huy refactored the query."** *(Chúng tôi sửa được lỗi vì Huy đã refactor câu query).*
4.  **"We resolved the issue because of Huy's query refactoring."** *(Chúng tôi sửa được lỗi nhờ việc refactor query của Huy).*
5.  **"The pipeline broke because a configuration file was missing."** *(Pipeline bị lỗi vì thiếu một file cấu hình).*
6.  **"The pipeline broke due to a missing configuration file."** *(Pipeline bị lỗi do thiếu file cấu hình).*
7.  **"The build failed because node modules were outdated."** *(Bản build thất bại vì các node module đã quá cũ).*
8.  **"The build failed because of outdated node modules."**
9.  **"Our traffic spiked because the marketing campaign launched."** *(Traffic tăng vọt vì chiến dịch marketing bắt đầu chạy).*
10. **"Our traffic spiked due to the launch of the marketing campaign."** *(Traffic tăng vọt do chiến dịch marketing được chạy).*

## 5. Bảng Tổng Hợp Từ Nối Nhanh

| Ý nghĩa liên kết | Liên từ (Dùng trong 1 câu) | Từ nối (Bắt đầu câu mới) | Giới từ nguyên nhân (Đi với Danh từ) |
| :--- | :--- | :--- | :--- |
| **Đối lập (Contrast)** | `, but` | `. However,` | `Despite` / `In spite of` |
| **Kết quả (Result)** | `, so` | `. Therefore,` / `. As a result,` | |
| **Nguyên nhân (Cause)** | `because` / `since` / `as` | | `due to` / `because of` |
| **Thêm thông tin (Addition)** | `, and` | `. Also,` / `. In addition,` | |
